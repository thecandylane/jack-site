# Jack Schliewe — Professional Site

Personal business website for full-stack development services, AI consulting, and technical education.

Built with **Next.js 15**, **React 19**, deployed on **Vercel**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub → Connect to [vercel.com](https://vercel.com) → Auto-deploys on every push.

## Stack

- Next.js 15 (App Router)
- React 19
- Google Fonts (Playfair Display, JetBrains Mono, DM Sans)
- Vercel hosting
